## 💡 teachtalk란?

<b>teachtalk</b>는 프로젝트 기반 멘토링을 통해 멘토와 멘티가 함께 성장하며, IT 산업의 기술 수준 향상과 지속 가능한 성장을 도모하는 <b>멘토링 중심의 경력 개발 및 취업 성공 플랫폼</b>입니다.
<b>취업 준비생과 경력 개발에 고민하는 개발자들</b>에게 실질적인 도움을 제공하며, IT 분야에서 경력과 실력을 쌓을 수 있도록 지원합니다. 👨‍💻👩‍💻
